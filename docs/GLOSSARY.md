# 📖 Thuật Ngữ Dự Án (Glossary) — NeoBoard

> Danh sách thuật ngữ sử dụng trong dự án. AI agent và member mới đọc file này để hiểu đúng nghĩa.

---

## A

| Thuật ngữ | Giải thích |
|---|---|
| **Access Token** | JWT token ngắn hạn (60 phút) dùng để xác thực API requests |
| **Activity Log** | Nhật ký hoạt động của user trong hệ thống |
| **Announcement** | Thông báo chính thức từ ban lãnh đạo/admin đến toàn bộ nhân viên |
| **API Response** | Cấu trúc JSON chuẩn mà backend trả về: `{ success, data, message, errors }` |

## B-C

| Thuật ngữ | Giải thích |
|---|---|
| **Badge** | Huy hiệu trong hệ thống gamification |
| **CRUD** | Create, Read, Update, Delete — 4 thao tác cơ bản với dữ liệu |
| **CRM** | Customer Relationship Management — Quản lý quan hệ khách hàng |

## D-E

| Thuật ngữ | Giải thích |
|---|---|
| **Dashboard** | Trang tổng quan hiển thị stats, charts, KPIs |
| **Department** | Phòng ban trong công ty |
| **DTO** | Data Transfer Object — Object chuyển dữ liệu giữa các tầng |
| **ERP** | Enterprise Resource Planning — Quản lý tài nguyên doanh nghiệp |

## F-H

| Thuật ngữ | Giải thích |
|---|---|
| **Feed** | Dòng bài viết hiển thị trong Timeline |
| **File Attachment** | File đính kèm trong post, announcement, hoặc survey |
| **Health Check** | Endpoint kiểm tra trạng thái hoạt động của hệ thống |

## I-L

| Thuật ngữ | Giải thích |
|---|---|
| **JWT** | JSON Web Token — chuẩn xác thực stateless |
| **Leaderboard** | Bảng xếp hạng trong gamification |

## M-P

| Thuật ngữ | Giải thích |
|---|---|
| **Migration** | Thay đổi schema database qua EF Core code-first |
| **My Page** | Trang cá nhân của user (profile, settings, activities) |
| **NeoBoard** | Tên dự án — Nền tảng dashboard quản trị doanh nghiệp |
| **Notification** | Thông báo in-app cho user (like, comment, announcement mới...) |
| **Pagination** | Phân trang dữ liệu: `page`, `pageSize`, `totalCount` |
| **Post** | Bài viết trong Timeline feed |
| **Priority** | Mức độ ưu tiên: Normal (0), Important (1), Urgent (2) |
| **Publish** | Hành động công bố announcement/survey từ draft ra cho mọi người xem |

## R-S

| Thuật ngữ | Giải thích |
|---|---|
| **RBAC** | Role-Based Access Control — Phân quyền theo vai trò |
| **Refresh Token** | Token dài hạn (7 ngày) dùng để cấp access token mới |
| **Role** | Vai trò user: SuperAdmin, Admin, Manager, Employee |
| **SaaS** | Software as a Service — Phần mềm cung cấp qua internet |
| **Seed Data** | Dữ liệu mẫu được tự động tạo khi init database |
| **Sidebar** | Menu điều hướng bên trái giao diện |
| **Soft Delete** | Xóa mềm — set `IsActive = false` thay vì xóa thật |
| **Survey** | Khảo sát nội bộ với nhiều loại câu hỏi |

## T-Z

| Thuật ngữ | Giải thích |
|---|---|
| **Timeline** | Feed bài viết nội bộ (giống Facebook wall nội bộ) |
| **Toast** | Notification popup ngắn ở góc màn hình |
| **Widget** | Thành phần nhỏ hiển thị thông tin trên dashboard |
